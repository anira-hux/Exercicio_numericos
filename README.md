### Integral Númerica 
- Exercício da primeira aula de numéricos com o João no primeiro semestre. 
- O objetivo era programar um código em C++ que calculasse a integral númerica de uma função utilizanod pontos conhecidos atravez do metódo dos trapézios. (esses pontos precisavam ter o mesmo passo - sendo asima mesma distancia entre eles) 
- O seguinte código aproximou sin(x) para o intervalo 0 a pi com o número de subintervalos variáveis mediante escolha do usuário.

### Método de Simpson 
- Exercício da Segunda aula de númericos
- O Objetivo era programar um código em C que demonstrasse a aproximação de uma integral atrávez do método de simpson.
- De forma resumida, esse método busca aproximar a area utilizando parábolas.
